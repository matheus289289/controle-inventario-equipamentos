# 📦 Controle de Inventário de Equipamentos

Este projeto é uma ferramenta desenvolvida em Python para gerenciar ativos de TI, permitindo o cadastro, consulta e controle de dispositivos de rede e hardware.

### 🚀 Funcionalidades
- Cadastro de equipamentos (Switch, Roteador, Servidor, etc).
- Armazenamento em banco de dados local (SQLite).
- Interface para gestão de estoque e ativos.

### 🛠️ Tecnologias Utilizadas
- **Linguagem:** Python 3.x
- **Banco de Dados:** SQLite3

### 📋 Como rodar o projeto
1. Instale o Python no seu computador.
2. Clone este repositório.
3. Execute o comando: `python "Controle de equipamente_inventario.py"`
